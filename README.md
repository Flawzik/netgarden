

### 🌿 NetGarden

> Социальная сеть для контекстного нетворкинга и встреч «в моменте».  
> Карта намерений вместо бесконечного скроллинга.


## 🎯 О проекте

NetGarden помогает находить людей для оффлайн-встреч по намерению, времени и локации.  
Вместо афиши событий — живая карта пинов с ограниченным сроком жизни.


## 🚀 Быстрый старт

### Требования
- Node.js ≥ 18.x
- npm ≥ 9.x или pnpm ≥ 8.x
- PostgreSQL ≥ 14 + PostGIS
- Git
